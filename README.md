## Уcтановка

1.  Установите Node.js и npm (Node Package Manager).
2.  Клонируйте репозиторий:

    ```bash
    git clone https://github.com/m00npr1ce/frontback6    
    ```

3.  Установите зависимости:

    ```bash
    npm install
    ```

## Запуск

1.  Запустите сервер:

    ```bash
    node server.js
    ```

2.  Откройте в браузере `http://localhost:3000`.
